LIP - Laboratório de Iniciação a Programação

O que essa disciplina ensina? Introdução à Programação de Computadores. Linguagens e plataformas para programação de computadores. Desenvolvimento de protótipos. Simulação e interação com dispositivos físicos.

O que fiz nessa disciplina?Nessa disciplina eu fiz diversos projetos utlizando Scratch para criar jogos e o Tinkercad para aprender e utilizar o arduino.
Para ver os projetos criados no Scratch e no Tinkercad, clique no botão ao lado "Projetos - LIP".

AEDS I - Algoritmos e Estrutura de Dados I

O que essa disciplina ensina? Representação e armazenamento de dados. Manipulação e movimentação de dados em memória principal e secundária. Abstração de dados. Estruturas e abstração de controle. Modularização, encapsulamento e herança. Recursividade. Documentação e testes. Implementação em linguagem de programação.

O que fiz nessa disciplina? Nessa disciplina eu aprendi a linguagem de programação C e um pouco de C++, fiz diversos exercícios para praticar e aprender mais sobre lógica de programação e a programar em C/C++.
Para ver os projetos criados, clique no botão ao lado "Projetos - AEDS I".

DIW - Desenvolvimento de Interfaces Web

O que essa disciplina ensina? Os padrões web e a importância do seu uso na construção de aplicações Web. As principais linguagens e plataformas de programaçãode interfaces Web. Criar páginas web utilizando os padrões e as linguagens citadas. Apresenta as tendências e novas tecnologias relacionadas à Web e à Internet.

O que fiz nessa disciplina? Nessa disciplina eu aprendi a criar páginas Web utilizando HTML e CSS , e também para deixar os sites mais dinâmicos aprendi a utlizar JavaScript JavaScript e integrar via APIs. Nessa disciplina criei um site totalmente responsivo e com um layout de um portal, com a temática de filmes.
Para acessar o site que eu desenvolvi nessa disciplina, clique no botão ao lado "Projeto - DIW".

TIAW - Trabalho interdisciplinar: aplicações web

O que essa disciplina ensina? Desenvolvimento de uma aplicação web front end usando um processo incremental e iterativo. Identificar um problema real a ser resolvido. Detalhar a necessidade a partir da perspectiva do usuário. Descrever os requisitos funcionais e não funcionais. Propor soluções baseadas em software. Conduzir um projeto de maneira sistemática e com trabalho em equipe.

O que fiz nessa disciplina? Nessa disciplina eu aprendi a criar um projeto do zero, utilizando métodos como Design Thinking e a metodologia ágil scrum, eu e meu grupo criamos um site com o intuito de ajudar as pessoas que tem dificuldade financeira, por meio de artigos sobre Inteligência Financeira e por uma plataforma para gestão de dinheiro.
Para acessar esse site, clique no botão ao lado "Projeto - TIAW".
